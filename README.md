E-Commerce Website

Overview

This is a responsive e-commerce website built using HTML, CSS, JavaScript, and Bootstrap. The project simulates a modern online clothing store where users can browse products, view product details, manage their shopping cart, and complete the checkout process.

Features

Responsive design for desktop, tablet, and mobile devices
Product listing page with categories and filters
Product details page with images, sizes, colors, and quantity selection
Shopping cart functionality
Checkout page with shipping and payment forms
User login page
Modern and clean user interface

Technologies Used

HTML5
CSS3
JavaScript (ES6)
Bootstrap 5

Pages

Home
Shop
sales
Checkout
Login
About Us
Contact

Purpose

The purpose of this project is to practice front-end web development concepts, including responsive layouts, user interface design, Bootstrap components, and JavaScript functionality commonly used in e-commerce applications.
